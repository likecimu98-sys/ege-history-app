ГДЕ ЛЕЖИТ ПРОГРАММА
───────────────────
Рабочий стол → ege-history-app-main → tools → fipi-parser

Полный путь:
C:\Users\Хозяин Саша\Desktop\ege-history-app-main\tools\fipi-parser

(Да, она внутри папки приложения по истории — так и должно быть.)


КАК ЗАПУСТИТЬ
─────────────
1. Зайдите в эту папку (fipi-parser).
2. Дважды кликните файл:  «Запустить парсер ФИПИ.bat»
3. Откроется чёрное окно (не закрывайте его!) и сам собой — браузер
   со страницей программы.
   Если браузер не открылся — откройте его вручную и введите в адресную
   строку:  http://localhost:3777

СОВЕТ: чтобы не искать файл каждый раз — кликните по
«Запустить парсер ФИПИ.bat» правой кнопкой → «Отправить» → «Рабочий стол
(создать ярлык)». Дальше запускаете с рабочего стола.


КАК СКАЧАТЬ ЗАДАНИЯ (4 шага на странице)
────────────────────────────────────────
1. ПРЕДМЕТ — выберите (по умолчанию уже стоит «История»).

2. ЧТО ВЫКАЧАТЬ — кликните нужные разделы (например «8. Великая
   Отечественная война»). Можно ничего не выбирать — тогда возьмётся всё.
   • Галочка «Также подобрать ответы» — если она нужна, поставьте её
     (подробнее внизу).

3. Нажмите «Найти задания», потом «Выкачать все найденные».
   Дождитесь зелёной полоски до конца.

4. Нажмите «Сохранить на компьютер». Появится кнопка «Открыть папку» —
   она покажет, куда всё сохранилось.

Если во время большой загрузки закрылось окно или пропало соединение:
1. Снова запустите «Запустить парсер ФИПИ.bat».
2. Вверху страницы появится «Незавершённая выгрузка».
3. Нажмите «Продолжить с места остановки».
Уже скачанные задания и уже найденные ответы повторяться не будут.


КУДА СОХРАНЯЕТСЯ
────────────────
В папку:  fipi-parser\output\<название темы и дата>\
Внутри:
  • задания.html — открыть двойным кликом, читать и печатать
  • задания.json и банк.json — для программ
  • папка images — картинки заданий

Задания по карте или письменному источнику сохраняются группой: общий
материал ставится один раз перед всеми относящимися к нему вопросами.
Для старых выгрузок эту информацию восстановить нельзя — их нужно скачать
заново новой версией парсера.

Если ФИПИ временно не отдал часть карт, перед сохранением появится кнопка
«Повторить загрузку карт». Она догружает только карты к блокам 9–12 и не
повторяет скачивание заданий или подбор уже найденных ответов.


ПРО ОТВЕТЫ (галочка «Также подобрать ответы»)
──────────────────────────────────────────────
• Ответы сайт ФИПИ не выдаёт — программа их ПОДБИРАЕТ. Это работает для
  заданий на соответствие, расстановку, последовательность и выбор.
• Для краткого текстового ответа (вписать слово) и сочинений — не работает.
• Это МЕДЛЕННО: одно задание на соответствие ≈ 20–30 секунд. Большая тема
  на 60 заданий может считаться 20–40 минут. Просто оставьте окно работать.


ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
────────────────────────
• Задания не загружаются / крутится вечно →
  выключите VPN (сайт ФИПИ не пускает зарубежные адреса) и попробуйте снова.
• «Не удалось запустить» в чёрном окне →
  не установлен Node.js. Скачайте с https://nodejs.org (кнопка LTS),
  установите, перезапустите .bat.


КОНСТРУКТОР ЗАДАНИЙ (сборка файла из скачанного)
─────────────────────────────────────────────────
На странице парсера вверху справа есть ссылка «🧩 Конструктор заданий».
Она делает готовый файл с заданиями из того, что вы УЖЕ скачали.

Как пользоваться:
1. Выберите экзамен (ЕГЭ/ОГЭ) и предмет.
2. Кликните номера заданий (например 1–7) — для Истории ЕГЭ номера
   определяются автоматически. Для обществознания — выбирайте по блоку/теме.
3. Выберите разделы (периоды истории / блоки обществознания) и, если нужно,
   тип ответа.
4. Нажмите «Предпросмотр» — покажет, как получится.
5. «Создать файлы» — сделает два файла в папке output/_подборки/:
   • подборка.html — открыть и нажать «Печать → Сохранить как PDF»
     (выглядит точь-в-точь как в банке, с картинками);
   • подборка.doc — открыть в Word и править.
   Ключ ответов — в конце файла.

Важно: если скачали новые задания — нажмите в конструкторе
«⟳ Обновить библиотеку», чтобы он их увидел.


КАК ЗАКРЫТЬ
───────────
Просто закройте чёрное окно. Всё скачанное останется в папке output.
